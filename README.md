
# 图书管理系统

参考项目：https://github.com/withstars/Books-Management-System <br>

#### 主要功能
1.  登录、注销、修改密码
2.  管理员对图书信息的增删改查、查看读者、查看借阅记录
3.  读者对图书信息的查看查询、修改个人信息、查看借阅记录
